> about me

I am proficient in the use of HTML, CSS and Bootstrap to create my websites. For coding I use VSCode as my code editor and use ChatGPT for minor assistance. For designing websites, I use Figma and Canva and for my illustrations I have used Clip Studio Paint, and Aseprite. I also have intermediate experience with PHP, SQL, MySQL, JavaScript, WordPress (Elementor) and Premiere Pro, along with some experience in computer support and troubleshooting issues using CMD.
